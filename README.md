# Typst-expert Extension for Gemini-code

> Contributions are Welcome!

## Installation

```bash
gemini extensions install https://github.com/kongdd/typst-llm-expert # from git
gemini extensions install local_repo_path # or from lcoal
```

```bash
# in gemini-code
> /extensions list

Installed extensions:
  typst-expert (v0.1.0) - active (not updatable)
```

## Usage

> 测试环境: gemini-3-pro-preview
